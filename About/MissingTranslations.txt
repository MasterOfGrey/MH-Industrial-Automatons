This file acts as a register for translation localisations needed for this mod.
The directory reference lists each individual line required to support it, including patched lines, and lists the Needed or Outdated language translations for languages that already have partial localisation.

If you would like to submit a localisation update for your language, please update this file accordingly.

Existing partially localised languages: German, Russian, Spanish, SpanishLatin
